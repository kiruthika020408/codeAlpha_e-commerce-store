# 🛒 E-Commerce Store Website

## 📌 Project Overview
This project is a simple **Full Stack E-Commerce Store Website** developed as part of my internship project.  
The application allows users to view products, add items to cart, manage cart items, and proceed with the order process.

## 🚀 Features
- 🏠 Home page with product listings
- 🛍️ Add products to cart
- ➕ Increase / decrease product quantity
- ❌ Remove products from cart
- 🧾 Cart summary with total price
- 🔐 User login page
- 📦 Order process
- 📱 Responsive design

## 🛠️ Technologies Used

### Frontend
     - HTML5
     - CSS3
     - JavaScript

### Backend
     - Node.js
     - Express.js

### Tools
     - Visual Studio Code
     - Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository


        git clone <your-github-link>


2. Open project folder


         cd CodeAlpha_EcommerceStore


3. Install dependencies


          npm install


4. Start the server


          node server.js


5. Open browser


         http://localhost:5000/
